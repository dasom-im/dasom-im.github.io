Dasom is a multilingual input method framework, which provides

  * Input Method Server
    * dasom-daemon including XIM

  * Client Modules
    * GTK+2, GTK+3, Qt4, Qt5

  * Indicators
    * GNOME shell: dasom agent extension
    * KDE, Unity, GNOME panel: dasom-indicator (using libappindicator3)
